Description:
-------------
This Java program is designed to simulate a binary search tree. 
On b1, its goal is to traverse the tree in its originality, then delete the 4th inserted node of the tree. Afterward, it will traverse the altered tree.
On b2, its goal is to travers the tree, delete duplicate nodes, then traverse it again.

Outputs:
--------------
Results describing actions taken and displaying tree traversals.
