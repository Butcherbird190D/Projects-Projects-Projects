
Description:
-------------
This C++ program allows the user to generate either an Arithmetic or a Geometric sequence.
The user is prompted for:
- The type of sequence (A or G)
- The initial term
- The common difference (for Arithmetic) or common ratio (for Geometric)
- The number of terms

The program validates all user inputs and generates the requested sequence.

-------------
- Enter 'A' or 'G' when asked for sequence type
- Enter any valid real number for initial term
- Enter any valid real number for difference or ratio
- Enter a positive integer for number of terms

Outputs:
--------------
A list of terms printed on a single line.
